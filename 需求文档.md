# AI 卡片管理系统需求文档

## 1. 项目概述

本项目是一个基于 Vue 3 的 AI 卡片管理系统，用于管理通过 OpenClaw 生成的内容，为用户提供一个直观、高效的创作管理平台。

## 2. 核心功能

### 2.1 卡片管理
- **创建卡片**：支持输入标题、描述、内容和标签
- **查看卡片列表**：展示所有卡片，支持点击查看详情
- **卡片详情**：查看卡片完整内容，支持重新生成（对接 OpenClaw）
- **删除卡片**：移除不需要的卡片

### 2.2 统计概览
- **总卡片数**：显示系统中卡片的总数
- **新增卡片**：显示近期新增的卡片数量
- **AI 对接状态**：显示与 OpenClaw 的连接状态

### 2.3 OpenClaw 集成
- **内容生成**：通过 OpenClaw API 生成卡片内容
- **重新生成**：对现有卡片重新生成内容
- **生成历史**：记录每次生成的历史版本

## 3. 技术架构

### 3.1 前端技术栈
- **框架**：Vue 3
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **构建工具**：Vite
- **样式**：原生 CSS

### 3.2 数据结构

#### 卡片数据结构
```javascript
{
  id: String,           // 卡片唯一标识
  title: String,        // 卡片标题
  description: String,  // 卡片描述
  content: String,      // 卡片内容
  tags: Array,          // 标签列表
  createdAt: String,    // 创建时间
  updatedAt: String,    // 更新时间
  generatedBy: String,  // 生成来源
  versions: Array       // 历史版本
}
```

## 4. 页面设计

### 4.1 首页
- 系统标题和副标题
- 统计卡片（总卡片数、新增卡片、AI 对接状态）
- 快捷操作按钮（创建卡片、查看全部）

### 4.2 卡片列表页
- 卡片网格展示
- 每个卡片显示标题、描述、创建时间
- 支持点击查看详情
- 支持删除操作

### 4.3 创建卡片页
- 表单输入（标题、描述、内容、标签）
- 提交和取消按钮

### 4.4 卡片详情页
- 卡片标题和元数据（创建时间、标签）
- 卡片内容展示
- 操作按钮（重新生成、删除）

## 5. 后续扩展

### 5.1 功能扩展
- **用户认证**：支持多用户登录
- **内容分类**：按标签或类别分类卡片
- **搜索功能**：支持按关键词搜索卡片
- **导出功能**：导出卡片内容为不同格式

### 5.2 技术扩展
- **持久化存储**：对接后端 API 或使用 localStorage
- **AI 模型选择**：支持选择不同的 AI 模型
- **批量操作**：支持批量生成、删除卡片
- **版本控制**：完整的版本历史管理
